# 🎤 Voice Input Simulator for Keyboard

## 📌 Description
Voice Input Simulator for Keyboard is a Python-based project that captures voice input from the user and converts it into text. The system then simulates keyboard typing by automatically entering the recognized text. This project helps improve typing efficiency and accessibility.

## 🚀 Features
- Convert voice input into text
- Simulate keyboard typing automatically
- Easy to use interface
- Improves typing speed
- Useful for accessibility and hands-free typing

## 🛠️ Technologies Used
- Python
- Speech Recognition Library
- PyAutoGUI (for keyboard simulation)
